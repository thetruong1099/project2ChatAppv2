# project2ChatAppv2
Chat App <br>
Mô tả <br>
Các chức năng<br>
<p>- user sử dụng kết bạn nhắn tin cho nhau.<br></p>
<p>- user tạo, chỉnh sửa profile.<br></p>
<p>- Các thiết bị sử dụng từ Android 6.<br></p>
<p>Công nghệ sử dụng:<br></p>
<p>- Firebase.<br></p>
<p>- Thư viện UI: navigation, lottie.<br></p>
