# project2ChatAppv2
Chat App<br>
Mô tả: <br>
Các chức năng:<br>
user kết bạn nhắn tin cho nhau.<br>
user tạo nhóm chat.<br>
user tạo, chỉnh sửa profile.<br>
Các thiết bị sử dụng từ Android 6.<br>
Công nghệ sử dụng:<br>
Firebase.<br>
Thư viện UI: material, lottie.<br>
